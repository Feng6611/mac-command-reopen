Command Reopen is a lightweight macOS menu bar utility for reopening closed or minimized windows when switching apps with Command+Tab.

Contact

Email: fchen6611@gmail.com

Project: GitHub repository

Expected behavior

  • The app runs in the menu bar.
  • It does not replace the native Command+Tab switcher.
  • Finder and Dock are ignored.
  • You can exclude specific apps from reopen behavior in Settings.

Troubleshooting

  • If an app should not be reopened, add it to the excluded apps list.
  • If launch at login is not available, check your macOS version and login items settings.
  • If behavior seems inconsistent right after an app launches, wait briefly and try switching again. Recent launches are intentionally suppressed to avoid duplicate windows.

For App Review

This app uses standard macOS APIs such as NSWorkspace and window reopen heuristics. It does not request Accessibility or Screen Recording permissions.
